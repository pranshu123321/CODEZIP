# 🚀 CodeQuest – Full-Stack Problem Solving Platform

## 📖 Overview

CodeQuest is a full-stack web application that provides an interactive platform for programmers to practice coding problems, submit solutions, learn through video tutorials, and receive AI-powered assistance. The platform also includes a powerful admin dashboard for managing problems, tutorials, and users.

---

# ✨ Features

## 👨‍💻 User Features

* 📝 Solve coding problems online.
* ⚡ Submit solutions and view submission history.
* 🎥 Watch detailed video explanations for problems.
* 🤖 AI Chatbot to get coding assistance and guidance.
* 📊 Personal dashboard to track learning progress.
* 🎨 Multiple themes for a personalized experience.
* 🔐 Secure authentication (Login & Signup).

---

## 🛠️ Admin Features

* ➕ Add new coding problems.
* ✏️ Update existing problems and tutorials.
* ❌ Delete outdated content.
* 👥 Manage registered users.
* 📈 Monitor submissions and platform activity.

---

## 🌟 General Features

* Fully responsive design for desktop and mobile.
* Modern UI built with React and Tailwind CSS.
* Fast frontend powered by Vite.
* Efficient global state management using Redux.
* REST API integration between frontend and backend.
* Scalable backend architecture using Node.js and Express.js.

# ⚙️ Installation

## Prerequisites

Make sure the following are installed:

* Node.js (Latest LTS recommended)
* npm
* Redis (if your backend configuration requires it)

---

## Clone the Repository

```bash
git clone https://github.com/garvshas/pro.git
cd pro
```

---

# Backend Setup

```bash
cd backend
npm install
npm start
```

The backend server will start on the configured port.

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The frontend development server will start (usually at `http://localhost:5173`).

---

# 🚀 Usage

* Create an account or log in.
* Browse available coding problems.
* Submit your solutions.
* Watch solution videos to improve your understanding.
* Ask the AI chatbot for coding help.
* Track your progress through the user dashboard.
* Administrators can manage problems, videos, and users from the admin panel.
